# 🚀 HeroAi — Autonomous Content Growth Infrastructure

![HeroAi Logo](https://i.ibb.co/L6MhYm9/logo.png)

**HeroAi** is a high-performance, autonomous AI production ecosystem designed to dominate social media algorithms with zero human intervention. From viral trend discovery to 4K video rendering and smart distribution, HeroAi handles the entire content value chain.

---

## 🌟 Key Features

### 🧠 Master Brain Orchestration
A decentralized AI pipeline that autonomously handles:
- **Global Viral Intelligence (GVI):** Real-time trend matching based on your niche.
- **Digital Twin Synthesis:** Automated face and voice cloning for consistent branding.
- **Dynamic Scripting:** AI-generated scripts optimized for high retention.

### 📽️ Autonomous Production Hub
- **Multi-Cloud Rendering:** Seamless integration with FFmpeg for 4K video stitching.
- **Asset Auto-Gathering:** Intelligent B-roll and licensed music selection.
- **Viral Replicator:** One-click bulk generation (7 days of content in minutes).

### 📈 Smart Distribution
- **Peak-Hour Analytics:** AI autonomously decides when to post live.
- **Multi-Platform Support:** Native hooks for YouTube, Instagram, and TikTok.
- **50/50 Marketplace:** Revenue sharing model for model owners and creators.

## 🚀 Quick Start (Local Development)

### Prerequisites
- Node.js installed
- MongoDB installed (or use Atlas)

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/pranaycablo/heroai.git
   cd heroai
   ```

2. Install dependencies:
   ```bash
   npm run build
   ```

3. Configure Environment:
   Create a `.env` file in the `backend/` directory:
   ```env
   PORT=5000
   MONGO_URI=your_mongodb_connection_string
   JWT_SECRET=your_secret_key
   ```

4. Run the Engine:
   ```bash
   npm start
   ```

Visit: `http://localhost:5000`
