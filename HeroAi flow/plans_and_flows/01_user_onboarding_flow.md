# Flow 1: User Onboarding & Personalization (Digital DNA)

## 1. Initial Signup
- User signs up via OTP or Google Login.

## 2. Platform & Channel Connect
- AI asks: "Which platforms do you want to work on?" (YouTube, Instagram, Facebook, LinkedIn, X, TikTok, WhatsApp, etc.)
- User adds channel links for each selected platform.

## 3. Deep Channel Details (The Secret Sauce)
- AI asks for detailed information to understand the user's core intent:
  - What type of content is created?
  - What is the goal? (e.g., Sales, Engagement, Brand awareness)
  - What is the speaking style/tone?
  - Who is the exact audience?
- Target Audience location (India, USA, UK, Global).
- User Type Identification (Creator, Business, Influencer, Agency).
- Category Selection (Pre-defined like Comedy, News, or Custom user input).

## 4. Digital Clone Creation (Model & Voice)
- Video Model: User can upload their face/image OR use an AI-generated model.
- Voice Selection: User can upload a custom voice sample OR use Auto AI voice.
- Result: AI creates a digital clone of the user.

## 5. Content Behavior Setup
- Daily Requirement: How many full videos vs shorts?
- Posting Preferences: What time does the user usually post?

## 6. Output: AI Blueprint Ready
- Backend AI processes all inputs to generate a **"User Content Blueprint" (Digital DNA)**.
- AI analyzes the user's level and suggests a smart plan (Free for beginners, Pro for serious creators).
