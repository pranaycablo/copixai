# Flow 5: Multi-API Rotation & Admin Control

## 1. Modular AI Architecture
- CopixAI is a network of specialized AI modules:
  - **Script AI** (Titles, text, desc)
  - **Video AI** (Clips, scenes, merge)
  - **Voice AI** (Speech generation)
  - **Thumbnail AI** (Image generation)
  - **SEO, Ads, and Trend AIs**

## 2. Multi-API Key System (Cost & Stability Engine)
- System rotates APIs to save cost and ensure 100% uptime:
  - Sequence: Free API -> 2nd Free API -> Cheapest Paid API -> Premium Backup.
  - Failover: If an API fails, it auto-retries or switches to fallback. User never sees an error.

## 3. Plan-based API Usage
- **Free Plan:** Uses free APIs, stock videos, limited usage.
- **Pro Plan:** Mixes APIs for balanced quality and cost.
- **Premium Plan:** Uses the fastest, highest-quality premium APIs exclusively.

## 4. Admin Dashboard (Control Tower)
- Admin has absolute control with Zero Liability (System runs autonomously).
- **API Management:** Add/manage multiple API keys and set priorities.
- **Cost Control:** Set daily/monthly limits and a kill switch.
- **User Monitoring:** Track active users, plan usage, and performance.
- **Revenue Dashboard:** Track total income, plan sales, and affiliate payouts.
- **Ticket Panel:** Monitor user issues (AI handled vs Manual).
