# Flow 6: Auto Mode & Market Intelligence

## 1. Dual User Control System
- **Manual Mode:** User generates content, edits scripts, and posts manually at the suggested times.
- **Full Auto Mode (The Real Power):** 
  - User connects social media accounts (YouTube, Instagram, LinkedIn, etc.) via OAuth.
  - AI takes complete control of the loop: Creates -> Posts -> Tracks -> Improves -> Repeats.
  - User literally does nothing after setup.

## 2. AI Decision Engine & Safeguards
- AI decides what to make, when to post, and where to post based on analytics.
- **Smart Safeguards:** Admin sets strict limits (Daily API limits, content limits) to prevent run-away costs and protect the system.

## 3. Market Scan System (Live Intelligence)
- AI constantly observes the market:
  - Trending topics globally and locally.
  - Competitor content performance.
  - Changing audience behaviors.

## 4. Trend Injection & Gap Filling
- **Trend Injection:** AI detects a viral topic (e.g., "Election") and injects it into the user's specific niche (e.g., creating a "Comedy Political Video").
- **Smart Opportunity Detection:** AI finds low-competition, high-demand topics and pushes the user to create content on them.
